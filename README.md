# ssky - Simple Bluesky Client

## Introduction

### Requirements

* Python 3.12 and later

### Install

```bash
pip install git+https://github.com/mkyutani/ssky.git
```

## Quick Start

Command line interfaces are formatted by `ssky function [options]`,
while function is one of the functions: timeline, post, profile, etc.
and options depends on the function.

### Timeline

```bash
SSKY_USERNAME=user.bsky.social SSKY_PASSWORD=xxxx-xxxxx-xxxx-xxxx ssky timeline
```
